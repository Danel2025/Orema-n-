## Notes pour la Migration

1. **Codes-barres** : Les codes commençant par `6916` sont fictifs pour les produits locaux. Vérifier les vrais codes EAN.

2. **Prix** : Basés sur les prix du marché 2024-2025. À ajuster selon l'établissement.

3. **Catégories imprimantes** :
   - Bar : Boissons, cocktails, cafés
   - Cuisine : Plats, entrées, desserts
   - Les deux : Certains établissements

4. **Stock** :
   - Boissons → géré (bouteilles comptées)
   - Plats → généralement non géré (préparation à la demande)
   - Accompagnements → optionnel

5. **Produits composites** :
   - Cocktails = spiritueux + sodas + garnitures
   - Menus = entrée + plat + dessert + boisson
